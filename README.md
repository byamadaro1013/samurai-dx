# Samurai DX

This is a Next.js project based on the ai-komon-lp architecture.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:8001](http://localhost:8001) with your browser to see the result.

## Scripts

- `npm run dev` - Start the development server on port 8001
- `npm run build` - Build the production application
- `npm start` - Start the production server on port 8001
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier
- `npm run format:check` - Check code formatting
